## Packages
recharts | For progress dashboard graphs and analytics
date-fns | For date formatting and manipulation
framer-motion | For page transitions and smooth animations
react-day-picker | For the calendar view in the journal
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
Authentication is handled via Replit Auth (blueprint:javascript_log_in_with_replit).
Use `credentials: "include"` for all fetch requests to ensure cookies are sent.
The app uses a dark theme by default.
AI endpoints return generated content - handle loading states gracefully.
